Instruction : 
How to setup : 
- sudo apt update
- sudo apt install python3-tk python3-pil.imagetk
- sudo apt install python3-pip
- /bin/python -m pip install opencv-python
- git submodule update --init --recursive 
- pip install torch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1
- pip install request
- pip install filterpy


- follow this GitHub : https://github.com/Qengineering/YoloV6-NPU 

dependencies for rknn :
-  sudo apt-get update
- sudo apt-get install cmake
- pip3 install rknn-toolkit2

follow this if you want to convert onnx to rknn : 
https://github.com/airockchip/rknn_model_zoo/tree/main/examples/yolov6


Follow this for setting up crontab (automatic boot):
- On your PC linux "crontab -e"
- add this line "@reboot (sleep 15 && export DISPLAY=:0 && cd /home/orangepi/github/fishcounter/src/ && /usr/bin/python3 launcher_app.py) >> /home/orangepi/github/fishcounter/launcher.log 2>&1"
#make sure your path is correct!
