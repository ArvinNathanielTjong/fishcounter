Instruction : 
How to setup : 
- sudo apt update
- sudo apt install python3-tk python3-pil.imagetk
- sudo apt install python3-pip
- /bin/python -m pip install opencv-python
- git submodule update --init --recursive 
- pip install torch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1
- pip install request
- pip install filterpy


- follow this GitHub : https://github.com/Qengineering/YoloV6-NPU 

dependencies for rknn
-  sudo apt-get update
- sudo apt-get install cmake
- pip3 install rknn-toolkit2

follow this if you want to convert onnx to rknn : 
https://github.com/airockchip/rknn_model_zoo/tree/main/examples/yolov6

for running the GPIO of the opi (buzzer) : 
- sudo apt update
- sudo apt install python3-periphery
- sudo nano /etc/udev/rules.d/99-gpio.rules
isi dengan : 
SUBSYSTEM=="gpio", KERNEL=="gpiochip*", ACTION=="add", PROGRAM="/bin/sh -c 'chown root:gpio /sys/class/gpio/export /sys/class/gpio/unexport ; chmod 220 /sys/class/gpio/export /sys/class/gpio/unexport'"
SUBSYSTEM=="gpio", KERNEL=="gpio*", ACTION=="add", PROGRAM="/bin/sh -c 'chown root:gpio /sys%p/active_low /sys%p/direction /sys%p/edge /sys%p/value ; chmod 660 /sys%p/active_low /sys%p/direction /sys%p/edge /sys%p/value'"

simpan dan keluar.

- sudo groupadd gpio   # Abaikan jika sudah ada
- sudo usermod -aG gpio orangepi
- sudo udevadm control --reload-rules
- sudo udevadm trigger
